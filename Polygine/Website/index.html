About Our Project

Jump and Run Adventure Game

Our team created Jump and Run Game, using HTML, CSS and JavaScript. You have to skip obstacles: mushrooms and rocks. You have to use Space button to jump. An indicator counts your score and shows it when the game is over.


Rules

The rules of the game are not very complicated. You are a little dwarf who is in the forest and has to overcome obstacles like mushrooms and stones without touching them, because otherwise he dies. By pressing "space" you can skip them and continue the game.
